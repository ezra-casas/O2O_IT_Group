# O2O_IT_Group


Free and/or cheap resources for IT Certifications (updated as of 2026). Focus on high-quality, mostly free video courses, official platforms, practice materials, and low-cost options. Exam fees are separate but many vendors offer discounts/vouchers.
## CompTIA
### A+ (220-1201/220-1202)

https://www.professormesser.com/ (Core 1 & Core 2 sections)
YouTube playlist example: Search "Professor Messer A+ 220-1201/220-1202" or use provided link.
* Original/Additional playlist: https://www.youtube.com/watch?v=AIfIA7hEgrw&list=PLG49S3nxzAnnes8ZGI-OBlKEukHCX46N8

#### Other Valuable Free/Low-Cost Certs & Resources (2026)

* Fortinet NSE 1-3, Cisco Networking Academy badges, Oracle Cloud Foundations, IBM SkillsBuild — often completely free with badges.
* Practice Questions: OpenExamPrep.com (96k+ free questions across CompTIA, AWS, Azure, Cisco).
* General Platforms:
  * Cisco Networking Academy (networking, cyber, Python).
  * Microsoft Learn + Virtual Training Days (occasional free exam vouchers).
  * Professor Messer (CompTIA focus).
  * YouTube channels: Jeremy's IT Lab, NetworkChuck, David Bombal (CCNA/cloud).


##### Tips for 2026:

Start with vendor free training (AWS Skill Builder, Microsoft Learn, Google Cloud Skills Boost, Cisco NetAcad) — many include hands-on labs.
Combine videos + practice exams + labs for best results.
Check for exam vouchers via events, military programs, or employer reimbursements.
Exam costs: CompTIA ~$300–$400 each; CCNA ~$300; Cloud fundamentals ~$100–$150 (discounts common).
